Declaración de accesibilidad

¿Por qué este es un sitio accesible?

Este sitio fue diseñado para ser utilizado de manera eficiente por todas las personas, sin importar la tecnología que posea, su idioma, cultura, ubicación geográfica o sus habilidades físicas o mentales.

Fue construido acorde los estándares de accesibilidad planteados por las Pautas de Accesibilidad Web (WCAG) 2.0 de la W3C, que permiten ampliar las opciones de accesibilidad a personas con diferentes posibilidades de audición, motricidad, vista y habilidad cognitiva.

Objetivos

  • Proporcionar diferentes posibilidades de uso del sitio para permitir el acceso a un mayor número de usuarios teniendo en cuenta diferentes formas de navegación o contexto de uso.
  • Optimizar la estructura del contenido para que el usuario pueda acceder al mismo de manera eficiente y eficaz.
  • Mejorar la experiencia del usuario gracias al desarrollo del diseño centrado en el usuario.

Niveles de conformidad

La estructura del sitio se encuentra en conformidad con la marca AA de las pautas arriba señaladas. Los requisitos de esta marca fueron comprobados con un análisis manual de la accesibilidad a través de diferentes herramientas semi-automáticas.

Características del sitio

  • Tecnologías: Estándares HTML5 y CSS3 (contenido y presentación separados)
  • Diseño y maquetación: Contraste entre los clores de texto y fondo, uso de unidades relativas (que permiten redimensionar el texto), alternativa a contenido no textual, navegación a través de teclado, navegación a través del lector de pantalla

Atajos de teclado

  • Ir al contenido principal de una página (Atajo de teclado: 1)
  • Ir al mapa de sitio (Atajo de teclado: 2)
  • Ir a la declaración de accesibilidad (Atajo de teclado: 3)
  • Ir a ayuda y preguntas frecuentes (Atajo de teclado: 4)

Uso de las teclas de acceso

  • Internet Explorer (Windows): ALT + Tecla + ENTER
  • Internet Explorer (Macintosh): Control + Tecla + ENTER
  • Mozilla (Windows): ALT + Tecla
  • Mozilla (Macintosh): Control + Tecla
  • Mozilla (Linux): ALT + Tecla
  • Opera 7.0 (Windows): MAYÚSCULAS + ESC + Tecla
  • Opera 7.0 (Macintosh): MAYÚSCULAS + ESC + Tecla
  • Opera 7.0 (Linux): MAYÚSCULAS + ESC + Tecla
  • Safari 1.2 (Macintosh): Control + ESC + Tecla
  • Konqueror 3.3 (Macintosh): Control + Tecla

Validadores utilizados

  • eXaminator
  • Validador W3C
